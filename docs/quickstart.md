# Quickstart

## 1) Install

```bash
make install-dev
```

## 2) Refresh dashboard and registry

```bash
make dashboard-refresh
```

## 3) Build main and ablation outputs

```bash
make main-results
make ablation-results
```

## 4) Optional: regenerate via scripts

```bash
bash scripts/reproduce_main_results.sh
bash scripts/reproduce_ablation.sh
```
